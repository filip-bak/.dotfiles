# Dotfiles 📁

This repo is for managing dotfiles using **GNU Stow**, allowing to keep configurations in one location and symlink them into home directory.

---

## 🚀 Quick Start
1. `cd ~`
1. git clone <repository-url>
1. `cd ~/.dotfiles`
1. `stow nvim`

## 🧹 Remove Links
1. `cd ~/.dotfiles`
1. `stow -D nvim`
